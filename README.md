# data-Cleaning-in-excel-
Sales Data Cleaning in Excel
This project demonstrates data cleaning and preprocessing on a sales details dataset using Microsoft Excel.

Tasks Performed:
Identified and handled missing values using Excel filters.
Removed duplicate records using the Remove Duplicates feature.
Standardized text values (e.g., gender, country names) for consistency.
Converted date values to a uniform DD-MM-YYYY format.
Renamed column headers to a clean and consistent format.
Checked and corrected data types (numbers, dates, and text) where required.

Tools Used:
Microsoft Excel

Outcome: A clean, consistent, and analysis-ready sales dataset suitable for reporting and data analysis.
